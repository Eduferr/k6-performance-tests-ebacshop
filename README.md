# K6 Performance Tests --- EBAC Shop

Este repositório contém um conjunto de testes de performance escritos em
**k6** para validar a estabilidade, tempo de resposta e comportamento
dos endpoints da API **EBAC Shop**.\
A estrutura segue um padrão modular que separa **simulações**,
**requisições**, **dados externos** e **utilitários**, tornando o
projeto organizado, reutilizável e fácil de manter.
